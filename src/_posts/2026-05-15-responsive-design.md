## 移动优先策略

移动优先（Mobile First）是现代响应式设计的核心思想。先为小屏幕设计基础样式，再通过媒体查询逐步增强大屏体验。

## 弹性布局方案

- **CSS Grid**：适合二维布局，如文章列表、分类网格
- **Flexbox**：适合一维布局，如导航栏、卡片内容
- **相对单位**：使用rem、em、%代替固定像素值

## 断点选择

合理的断点应该基于内容，而非设备：

```css
/* Mobile: < 768px */
/* Tablet: 768px - 1199px */
/* Desktop: >= 1200px */
```

## 响应式图片

使用`srcset`和`sizes`属性，根据视口宽度加载合适尺寸的图片：

```html
<img src="small.jpg"
     srcset="medium.jpg 768w, large.jpg 1200w"
     sizes="(max-width: 768px) 100vw, 50vw">
```

## 总结

响应式设计不是一次性工作，而是一个持续优化的过程。关注内容、渐进增强，才能打造出真正优秀的跨设备体验。
